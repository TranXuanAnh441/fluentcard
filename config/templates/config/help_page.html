Welcome to FluentCard, a ChatGPT-powered Flashcard Website !

Let's start with adding your card into deck

  • First, let's create new deck by 'Add Deck' button.
  • Then add the word you want to learn by 'Add Card' button.
  • Search for any word you want to learn on our built in dictionary system.
  • Then, let's learn the deck by 'Learn Deck' button and answer the questions for each word.
  • Keep on learning everyday and check your personal progress on the 'Progress' page.

Let's search new words on 'Dictionary' page

  • Vocabulary source from Jisho.org API
  • Search for any word you want to learn in both English and Japanese.
  • Check definitions, example sentences with images and audios.
  • Add the word directly to the deck and start learning without wasting time creating flashcards.

Let's practice using the words in communication with 'RolePlay' page.

  • You can choose the provided prompt or create your own prompt from the list.
  • Click on the prompt to start the conversation.
  • Try to use the word you are learning as much as possible. Your word use will be recorded in the progress page.
  • End the conversation and check the review by hitting the 'End' button.

Note: About the Timeout error:

This chat use generative AI, so the time to generate might leads to timeout error. Please understand and use the application in good internet connection to have the best experience.
